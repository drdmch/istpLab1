# Лабораторний проєкт №1: Розробка вебдодатку "ITschoolMVC"

## Предметна область: Онлайн-школа програмування

### 1. Постановка задачі
Даний проєкт є інформаційною системою для керування процесом дистанційного навчання в онлайн-школі програмування. Система забезпечує взаємодію між трьома типами користувачів: Гостями, Студентами та Адміністраторами.

**Основні можливості системи:**
* **Для гостей:** перегляд каталогу курсів, пошук програм навчання та ознайомлення з інформацією про школу.
* **Для студентів:** реєстрація та авторизація, керування особистим профілем, запис на курси та перегляд навчальних відеоуроків.
* **Для адміністраторів:** повне керування структурою курсів (створення, редагування, видалення), модерація уроків, зміна вартості навчання та експорт аналітичних даних про історію цін.

---


### 2. Технологічний стек
* **Мова програмування:** C# (.NET 9.0)
* **Архітектурний шаблон:** MVC 
* **ORM:** Entity Framework Core 
* **База даних:** PostgreSQL
* **IDE:** JetBrains Rider (macOS)
